## Restricciones por rol

- `cashier`: solo ve placa, tiempo, monto y estado de pago.
- `security`: puede consultar incidentes y estado operativo.
- `auditor`: solo puede consultar auditoria.
- No se muestran fotos, embeddings ni evidencia biometrica completa.
